Usage
=====

Used to specify where immigrants spawn

Example from immigration.txt
============================

`spawnLocation = `**`NearLeader`**

Comments
========

-   It is possible to not have a fixed spawn location, and minions will
    come in from the edge of the map.

Valid Values
============

-   NearLeader
-   Pregnancy
-   FurnitureType
